The Dashboard for Coffee Shop Sales Analysis is made with full use of Excel’s analytical tools to organize, explore, and visualize the data effectively

Data Processing
Data cleaning in Power Editor, new columns were made and some were processed

Pivot Tables
Created Pivot Tables to explore trends and summarize the data. 

Analyzing sales by month, store, and product category

Identified top-selling items Pivot tables gave us the flexibility to rearrange data


Data Model Integration
To manage large data and create dynamic relationships, Excel’s Data Model is utilised. By adding the data to the model:

We could relate different fields efficiently

It improved performance when handling large volumes of transactions

We used measures like total revenue, average transaction value, and more with DAX formulas

Slicers
To enhance interactivity and make  analysis more user-friendly:

Slicers are implemented on fields like store location, product category, and month


Dashboard Creation
Using the pivot tables, charts, and slicers, we designed a visual dashboard that provides a snapshot of key metrics.
